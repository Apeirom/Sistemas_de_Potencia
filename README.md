# Análise Computacional de Sistemas de Potência

Este repositório contém o conjunto de scripts em Python desenvolvidos para a disciplina de **Sistemas de Potência (2026)**. O objetivo é automatizar a resolução de circuitos trifásicos complexos, cobrindo cenários de equilíbrio, faltas assimétricas, desequilíbrio total e modelagem em valores por unidade (p.u.).

## 📂 Estrutura do Repositório

- `item_1.py`: Análise de sistemas trifásicos equilibrados (Cálculo de correntes, potências complexas e correção de fator de potência).
- `item_2.py`: Análise de faltas assimétricas (Rompimento de fase B e cálculo de deslocamento de neutro).
- `item_3.py`: Análise de sistemas desequilibrados com acoplamento mútuo e aterramento por impedância (Questão de alta complexidade).
- `item_4.py`: Simulação de fontes assimétricas em redes simétricas (Tensões de geradora desequilibradas).
- `item_5.py`: Modelagem de circuitos monofásicos em Valores por Unidade (p.u.) com mudança de base.

## 🚀 Como Utilizar

### Pré-requisitos
Certifique-se de ter o Python 3.x instalado em sua máquina. O código utiliza apenas a biblioteca padrão `cmath` e `math`, não sendo necessária a instalação de pacotes externos.
